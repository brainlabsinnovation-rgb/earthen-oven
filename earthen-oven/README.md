# Earthen Oven - Website

A premium fine dining website for Earthen Oven, located on the 14th floor of Fortune Landmark, Ahmedabad.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod

## Features
- **Immersive Hero Section:** Parallax effects with 14th-floor rooftop views.
- **Menu Showcase:** Categorized menu with signature dish highlighting.
- **Reservation System:** Premium form with date/time selection and validation.
- **Private Events:** Dedicated page for corporate and private bookings.
- **Gallery:** Lightbox gallery of the ambiance and views.
- **Live Music:** Section highlighting the evening Gazal performances.
- **Responsive Design:** Optimized for all devices with a mobile-first approach.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open:** [http://localhost:3000](http://localhost:3000)

## Project Structure
- `app/`: Next.js App Router pages
- `components/`: React components (UI, Sections)
- `lib/`: Utility functions
- `public/`: Static assets (images)

## Customization
- **Colors:** Defined in `app/globals.css` (primary: #8B6F47, accent: #D4AF37)
- **Fonts:** Playfair Display (Headings) & Lato (Body)
